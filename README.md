# Disaster Response & Resource Coordination Platform

## Overview

A real-time disaster management system that helps coordinate emergency response during floods, cyclones, earthquakes and other disasters.

## Features

- SOS Request Management
- Volunteer Matching
- Priority-Based Emergency Handling
- JWT Authentication
- Real-Time Alerts (WebSockets)
- Resource Tracking
- Admin Analytics Dashboard
- Google Maps Integration
- Heatmap Visualization

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Chart.js
- Tailwind CSS

### Backend
- Spring Boot
- Spring Security
- JWT
- WebSocket

### Database
- MySQL

## Architecture

Frontend → REST API → Spring Boot → MySQL

Frontend ← WebSocket → Spring Boot
